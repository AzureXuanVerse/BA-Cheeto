## 📝 Usage 
1. Set steam lunch option to `-force-d3d11` (Do this until dx12 backend is implemented)
2. Launch the game
3. Use your preferred injector and inject the DLL

## ✨ Features
- No Skill Cost
- Dumb Enemies (enemies ignore you)
